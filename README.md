<br/><br/>

Aplicacao full stack de chat em tempo real com foco em portfolio profissional. Permite criar conta, entrar em salas, enviar mensagens instantaneas, visualizar usuarios online e acompanhar historico recente das conversas, com backend Node.js e banco PostgreSQL.

<br/>

```bash
# Clone o repositorio
git clone https://github.com/EnzoPontoni/RealTimeChat.git

# Entre na pasta
cd RealTimeChat

# Backend
cd server
npm install
npx prisma generate
npm start

# Frontend (novo terminal)
cd ../client
npm install
npm run dev
```
